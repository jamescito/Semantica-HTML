Para que sirven las tablas HTML

.Las tablas en HTML sirve para organizar bien los datos es decir cuando se necesita que los datos se organicen en fila y columna ya que de otra manera dificilmente seria comprendida por el usuario. La creacion de las tablas no es nada distinto de las otras etiquetas en HTML ya que no es complicado a como podemos ver en la miniatura asi se construyen.

Iframe HTml

Iframe HTML con palabras sencillas es un marco en linea lo cual es de muy importante en las etiquetas HTML ya que este sirve para incrustar otro documento dentro del documento HTML actual con el que se esta trabajando en el index, al iframe tambien se le puede establecer altura y anchura segun sea la necesidad para que se mire de una forma correcta

Propiedad float

La propiedad float CSS su trabajo a realizar es especificar si un elemento puede salir del flujo normal o mover lo para que aparezca en la izquierda o derecha del contenedor todo con el fin que se pueda ver los texto a su alrededor.

FLEXBOX CSS

Para el diseño de flexbox o flexible box este sirve para el diseño de una pagina web que lo que hace es permitri colocar los elemento para que estos puedan comportarse de forma predecible segun la necesidad es decir que este pueda hacer a como dar los diferente tipo de pantalla en los diferentes dispositivos con lo que se pueda nevegar en la web es decir dispositivos inteligente

Diseño GRID CSS

Con el diseño grid lo que se hace es contener funsiones de diseños para una pagina web ya que con grid se puede llegar a obtener distinto diseño para establecerlo, tambien se puede lograr dividir la pagina en area o regiones principales.

el grid al igual que el Iframe permite un autor linear elementos en filas y columnas y la ventaja del grid es que con este se puede lograr mucho mas diseño de una forma sencilla con las tablas.
